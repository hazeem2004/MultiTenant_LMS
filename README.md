# MultiTenant_LMS (DevCohort)

DevCohort is a mobile-first application built with Flutter and Firebase, designed for independent coding bootcamps to manage curriculum and student submissions.

## Features
- Multi-tenancy support.
- Instructor-led cohorts and curriculum management.
- Markdown-supported assignments.
- Secure enrollment via invite tokens.
